# Tips Tracker (Accounting of Tips)

A simple console-based Python application for tracking daily tips.
The project was created as a learning exercise to practice Python basics,
working with files, dates, and program structure.

---

## Features

- Add tips for today
- Add tips for a past day
- View total tips for:
  - today
  - last 7 days
  - current month
- Show all saved tips
- Show tips statistics by weekday
- Find the best day of the week by tips amount
- Fun comments based on the entered tips amount

---

## How it works

All tips are stored in a text file (`tips.txt`) in the following format:


Example:
2026-01-05 100
2026-01-06 150


The program reads this file, processes the data, and shows statistics
based on user selection in the menu.

---

## Menu

1 - Add tips
2 - Add tips for past day
3 - Tips for today
4 - Tips for a week
5 - Tips for a month
6 - Show all tips
7 - Best day of the week
8 - Stats by weekday
0 - Exit
